# PostgreSQL Knowledge for TmaxTibero

## Quick start
1. **Get the code**
```sh
  git clone http://192.1.1.101/db/hypersql/postgresql-knowledge.git
 ```
2. **Install dependencies with yarn**
```sh
  npm install -g yarn
```
2.1 yarn version 확인
```sh
  yarn --version
```
2.2 yarn 설치
```sh
  yarn install
```

3. **Start the website server**
3-1. gatsby 설치
```sh
  yarn add gatsby
```
```sh
  gatsby develop
```
